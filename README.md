# C-Sharp Calculator

Простой консольный калькулятор, написанный на языке C# в рамках обучения.

## Описание
Программа позволяет выполнять основные арифметические операции:
* Сложение (`+`)
* Вычитание (`-`)
* Умножение (`*`)
* Деление (`/`)

Калькулятор поддерживает непрерывную работу в цикле до тех пор, пока пользователь не введет команду `exit`.

## Как использовать
1. Запустите приложение через терминал: `dotnet run`
2. После запуска введите два действительных числа через пробел (например: `10 5`).
3. Далее введите желаемую операцию (`+`, `-`, `*` или `/`).
4. Программа выведет результат вычислений.

## Пример ввода:
Write 2 numbers (double) in the folowing format: {number1} <space> {number2}

10 5

Choose operation: +, -, *, /

$+$

Operation result: 15
